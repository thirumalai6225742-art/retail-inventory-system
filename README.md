# Smart Retail Inventory System

A simple inventory management web app for tracking products, stock, sales, and revenue. The app uses browser local storage to persist data locally.

## Features
- Add, edit, delete, and sell products
- Search products by name
- Display inventory metrics such as total products, inventory value, low stock items, revenue, and top-selling product
- Login screen with a simple admin authentication flow

## Run locally
Open the project folder in a browser and start with the login page:
- login.html

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Push this project to the repository.
3. In GitHub, go to Settings > Pages.
4. Choose the main branch and the root folder as the source.
5. Your site will be available at:
   https://<your-username>.github.io/<your-repo-name>/
